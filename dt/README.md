## 功能特性

### 🎯 闭环工作流程

7步数据恢复闭环流程，完整的矩形环形设计：

```
        🔄 持续优化闭环
    ╔════════════════════════════╗
    ↓                            ↓
┌────────┐→┌────────┐→┌────────┐→┌────────┐
│ 需求接入│ │风险评估│ │介质防护│ │镜像复制│
└────────┘ └────────┘ └────────┘ └────┬───┘
    ↑                                 ↓
    ↑   ┌────────┐ ┌────────┐ ┌────────┐
    │   │保密销毁│←│验证交付│←│数据修复│
    │   └────────┘ └────────┘ └────────┘
    ╚════════════════════════════════╝
```

**特色效果**：
- ⚡ 电子流动动画（4秒循环）
- 💫 数字徽章脉动发光
- 🎯 4个转角方向箭头
- ✨ 天空蓝主题色

### 🧭 智能导航系统

参考 Ecopia AI 的导航技术：

- **下拉菜单**: 服务项目、解决方案、关于我们
- **自动高亮**: 根据滚动位置实时更新
- **响应式**: 桌面悬停/移动点击
- **毛玻璃效果**: 半透明背景 + 模糊

### 🤖 AI智能客服

模拟数据恢复专家的智能对话系统：

- 场景识别（RAID、数据库、硬盘等）
- 专业建议和解决方案
- 快速报价估算
- 恢复时间评估

## 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/wkdays/datarecover.git

# 进入目录
cd datarecover

# 启动本地服务器
python3 -m http.server 8000

# 访问网站
# 浏览器打开 http://localhost:8000
```

### 直接使用

也可以直接双击 `index.html` 在浏览器中打开。
